# Lakehouse Samples

This repository contains sample configurations for testing a Lakehouse. For testing purposes only. Not for production use.

## Sample A

Sample A uses:

- Apache Iceberg as a table format.
- Apache SeaweedFS as object storage.
- Apache Polaris as a catalog for Apache Iceberg.
- DuckDB and Spark SQL as query engines.

For more details, see [README](./sample-a/README.md) under the `sample-a` directory.
